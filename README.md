# Trip-cost-estimator

Trip Cost Estimator is a full-stack web application that helps users plan and manage trip expenses efficiently. Users can create trips, add expenses, calculate total and per-person cost, and securely share trips with others.

The project is built step-by-step from a simple React app to a complete system with authentication, authorization, and collaboration features.

# Tech Stack

# Frontend :

    React.js
    Redux Toolkit
    React Hook Form
    React Router
    Axios

# Backend :

    Node.js
    Express.js

# Database :

    MongoDB
    Mongoose

# Authentication & Security :

    JWT (JSON Web Token)
    bcrypt
    Multer (for file upload)

# ⚙️ Installation & Setup

1️⃣ Clone the repository
git clone <your-repo-url>
cd trip-cost-estimator
2️⃣ Setup Backend
cd backend
npm install

Create .env file:

PORT=5000
JWT_SECRET=your_secret_key

Run backend:
npm run dev

3️⃣ Setup Frontend
cd frontend
npm install
npm run dev
